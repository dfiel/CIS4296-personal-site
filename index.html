Greenwave Reality Smart Bulb System

I reverse engineered the API for a smart home system that was discontinued by the manufacturer so I could regain local control of my lightbulbs. I turned this API into a Python library, and then wrote a Home Assistant Component to wrap that library in a format that the Home Assistant ecosystem could understand.

VyOS BGP Simulation Project

As part of an Honors project for a Network Architectures class, I constructed a simulated "internet" with multiple routers using the Border Gateway Protocol (BGP) to distribute routes. I used the VyOS router operating system to achieve this. To create the necessary virtual routers, I used Vagrant to programatically define the virtual environment. To configure the routers once created by Vagrant, I used Ansible with roles and variables to automatically configure the routers according to a set of configuration files.

Temple Robotics 2021 Robot Code

I served as the Programming Team lead for Temple Robotics, an Engineering Student Organization that competes in NASA's Lunabotics Robotics Mining Competition. I lead the development of the code for last year's competition robot, which was written in Java utilizing the FIRST Robotics Competition control system. By the end of the season, we had 2,390 lines of code, or 3,566 if you include comments and whitespace. The code is private, but before the end of last year we were able to completely remote control the robot and receive various telemetry all from this codebase.
